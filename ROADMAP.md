# 🗺️ Roadmap — TAMA Dashboard

Ideas y mejoras futuras, agrupadas por prioridad. No es un compromiso de fechas.

## Corto plazo
- [ ] **Notificaciones por email** al asignar una tarea o acercarse una fecha límite.
- [ ] **Adjuntos reales** con Supabase Storage (ahora hay un campo de enlace).
- [ ] **Exportar** tareas y empresas a CSV / PDF.
- [ ] Notificaciones *in-app* (campana en el header) con la tabla `notifications`.
- [ ] Filtros guardables / vistas guardadas por usuario.

## Medio plazo
- [ ] **Integración con Google Calendar** (sincronizar fechas límite).
- [ ] **Integración con Slack / Discord** (avisos de cambios de estado).
- [ ] **Métricas de productividad**: tareas completadas por semana, tiempo medio por estado.
- [ ] Subtareas y checklists dentro de una tarea.
- [ ] Menciones `@usuario` en comentarios.

## Largo plazo
- [ ] **CRM avanzado**: pipeline de ventas con etapas.
- [ ] **Facturación** ligada a empresas y tareas.
- [ ] **Permisos avanzados por equipo / proyecto** (más allá de los 3 roles actuales).
- [ ] App móvil (PWA instalable o wrapper nativo).
- [ ] Time tracking por tarea.

## Deuda técnica / mejoras internas
- [ ] Tests end-to-end (Playwright).
- [ ] Migraciones versionadas de la base de datos.
- [ ] Modo offline con cola de sincronización.
